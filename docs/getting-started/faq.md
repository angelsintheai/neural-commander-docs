# Frequently Asked Questions

## What is Neural Commander?

Neural Commander is an AI project intelligence platform that preserves context across sessions, tracks requirements, and enhances developer productivity.

## Is it open source?

Yes, Neural Commander Community Edition is open source under Apache 2.0 license.

## Does it require internet?

No, NC runs entirely locally using Ollama for AI inference.

## What AI models does it support?

Any model available through Ollama, including Llama 3.2, CodeLlama, and more.
